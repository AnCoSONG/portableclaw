====================================================
  PortableClaw - OpenClaw Portable Edition
====================================================

QUICK START
-----------

  Windows:  Double-click  start.bat
  macOS:    Double-click  start.command
  Linux:    Run           ./start.sh

First run will guide you through setup (API keys, etc).
After that, Gateway starts and Dashboard opens in browser.

Dashboard: http://127.0.0.1:18789/


UPDATE
------

  Windows:  Double-click  update.bat
  macOS/Linux:  Run       ./update.sh


CLI (Advanced)
--------------

  Windows:      bin\openclaw.cmd <command>
  macOS/Linux:  bin/openclaw <command>

  Examples:
    bin/openclaw status
    bin/openclaw gateway status
    bin/openclaw onboard


UNINSTALL
---------

  Delete this folder. Config is in ~/.openclaw/
